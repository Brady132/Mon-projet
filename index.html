Les commandes git

  • Git add . Cette commande sert à ajouter les modifications d’un fichier a la zone d’index.
  • Git status Liste tous changements apportés à l’espace de travail.
  • Git reset Enlève les modifications d’un fichier de l'index, mais conserve son contenu.
  • Git commit -m Enregistre les modifications présentes dans la zone d’index dans l’historique du dépôt
  • Git log Montre l'historique des commits pour la branche courante.
